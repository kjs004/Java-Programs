# Duplicate Table Finder

## Idea Behind the Program

As we know, some products can occur in more than one multiplication table. For example:

12 × 3 = 36

18 × 2 = 36

Although these expressions belong to different tables, they produce the same result.

This program generates multiplication tables from 12 to 18 and compares the products obtained from different tables. Whenever the same product appears more than once, it identifies and displays the matching multiplication pairs.

The purpose of this program is to explore a mathematical pattern while practicing Java concepts such as loops, arrays, comparisons, and logical problem-solving.

## Sample Output

Duplicate Product = 36

12 x 3

18 x 2

Duplicate Product = 112

14 x 8

16 x 7

## Concepts Used

- Arrays
- Nested Loops
- Conditional Statements
- Pattern Detection
- Problem Solving

  ## Note

I came up with this idea while observing multiplication tables and noticing that the same product can appear in different tables.
